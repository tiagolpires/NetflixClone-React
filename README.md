# Netflix Ui-Clone 

Ui-Clone da Netflix para estudar React.Js

## Built With

 - [React.Js](https://reactjs.org/) - Configuração e componentização da Ui
 - [Tmdb - API](https://www.themoviedb.org/) - Data para construção da Ui
